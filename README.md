# Império Dashboard

## Requisitos

- **pnpm** (não usar npm/yarn)

## Configuração local

Crie um arquivo `.env` na raiz do projeto com:

```
VITE_SUPABASE_URL=https://cxhypcvdijqauaibcgyp.supabase.co
VITE_SUPABASE_ANON_KEY=COLE_SUA_ANON_KEY_AQUI
```

Referência: `docs/ENV_EXEMPLO.txt`.

## Rodar o projeto

Instalar dependências:

```
pnpm install
```

Subir dev server (**porta 3000**):

```
pnpm dev
```
